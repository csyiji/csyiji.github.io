一、Git简介

Git 是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何大小项目的版本管理。与 CVS 相比，它不是把版本库放在集中的存储服务器，而是每个开发者电脑上都是一个版本库，无须联网就可以管理代码版本，并且分支管理更高效，切换更快捷方便，已经成为现在开发者主流版本控制工具。
